# Study-Timetable-Planner
An AI-powered Study Timetable Generator built using Lyzr Agent Studio.
This agent helps students create personalized daily or weekly study schedules based on subjects, available study hours, exam deadlines, and subject difficulty, improving productivity and study efficiency.
